在ubuntu18.04下借助wine玩东方project游戏

发表于 2019-03-15 | 分类于 others

有时候在实验室偷偷懒会想玩一些windows平台的小游戏，但是实验室的电脑上只装了ubuntu系统，尝试过用虚拟机玩，但是体验比较差，有明显的延迟。后来了解到wine这个工具，这里记录一下使用wine来玩东方风神录的过程。
首先安装wine 

sudo apt-get update
sudp apt-get install wine-stable

然后安装winetricks

cp /home/wangyi/.wine/dosdevices/c:/windows/regedit.exe /home/wangyi/.wine/dosdevices/c:/windows/system32/
wget http://www.kegel.com/wine/winetricks

记着把路径中的wangyi改成自己对应的系统用户名….
接着配置为可执行 并移动到用户目录

chmod +x winetricks
mv winetricks /usr/local/bin

下一步启动winecfg来初始化wine,直接在终端输入winecfg
如果一切顺利的话，就可以使用wine thxx.exe来运行程序了…然而中途还是可能碰到一些问题，我在过程中碰到过两个问题。
问题一、winecfg报错，截图如下:

解决方法: su root 以root用户来运行就OK啦
问题二、wine thxx.exe后报错X Error of failed request: BadValue (integer parameter out of range for operation)
解决方法: 发现是自己显卡驱动装的有问题，重装后问题解决。
实际运行结果是这样的:

有一个小瑕疵就是标题文字乱码，但是游戏内的文字没有问题。除此以外十分流畅，游戏体验很棒。
据说乱码问题可以通过替换字体文件的方式解决，但游戏体验不受影响就懒得折腾了。
